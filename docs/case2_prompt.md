# Промпт — Кейс 2

```text
Тебе переданы два документа:

1. case2_citizen_appeals.md — поток из 7 обращений граждан, поступивших в Правительство Камчатского края.
2. case2_routing_regulation.md — регламент работы с обращениями, включающий перечень тематических категорий, уровни приоритета и правила маршрутизации.

Классифицируй каждое обращение в соответствии с регламентом. Для каждого обращения укажи:

1. Номер и дата обращения (из входящего номера).
2. Основная тематическая категория — одна из 10 категорий регламента (раздел 2). Если обращение затрагивает несколько тем, определи главную по правилам раздела 5.
3. Дополнительные категории (при наличии) — если обращение затрагивает смежные темы, перечисли их.
4. Ответственный исполнитель — наименование министерства/агентства, соответствующего основной категории.
5. Соисполнители (при наличии) — органы по дополнительным категориям.
6. Уровень приоритета — срочный, повышенный или стандартный, с указанием конкретного критерия из раздела 3 регламента, на основании которого определён приоритет.
7. Срок рассмотрения — количество дней в соответствии с присвоенным приоритетом.
8. Краткое обоснование — 1–2 предложения, почему выбрана именно эта категория и приоритет.

Результат оформи в виде структурированной таблицы или пронумерованного списка. После классификации всех обращений приведи краткую сводку: сколько обращений по каждой категории и каждому уровню приоритета.
```
