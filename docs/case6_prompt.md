# Промпт — Кейс 6

```text
Тебе переданы два документа:
- Финансовая модель проекта (case6_financial_model.md) — всесезонный туристический комплекс «Вилючинский».
- Регламент инвестиционного комитета (case6_committee_regulations.md) — порядок подготовки экспертного заключения, критерии принятия решения, особые условия.

Выполни работу в два шага.


Шаг 1. Проверка внутренней согласованности модели

Проверь финансовую модель на внутренние противоречия. Для каждой проверки покажи расчёт и вывод (ОК / расхождение):

1. CAPEX vs источники финансирования. Сумма собственных средств + кредит + субсидия должна быть равна итого CAPEX. Есть ли разница?

2. Выручка vs загрузка x тариф. Для первого года базового сценария рассчитай выручку: (номерной фонд x средняя загрузка x средневзвешенный тариф x 365 дней) x (1 + доля допдоходов). Сравни с цифрой в модели.

3. OPEX vs процент от выручки. Для каждого года базового сценария проверь, что OPEX составляет 62% от выручки. Отметь годы с отклонением более 1%.

4. Обслуживание долга. Рассчитай аннуитетный платёж для кредита 1 000 млн руб. под 14% на 8 лет. Совпадает ли с цифрой 195 млн руб./год в модели? Проверь аналогично для пессимистичного сценария (ставка 16%, платёж 210 млн руб./год).

5. Сумма непредвиденных расходов. Проверь, что 197 млн руб. действительно составляют 10% от суммы остальных статей CAPEX.

Сведи результаты в таблицу: проверка | ожидаемое значение | значение в модели | расхождение | вывод.


Шаг 2. Подготовка экспертного заключения для Комитета

Используя результаты шага 1 и требования регламента, подготовь экспертное заключение строго по формату из раздела 2.2 регламента:

1. Краткое описание проекта — не более 1 страницы.

2. Сравнительная таблица сценариев — NPV, IRR, срок окупаемости, пиковый кассовый разрыв, потребность в дополнительном финансировании для каждого из трёх сценариев.

3. Анализ чувствительности — рассчитай влияние на NPV и IRR базового сценария четырёх факторов:
   - ключевая ставка +2 п.п. (дисконтирование по 16% вместо 14%);
   - загрузка -10 п.п. по всем годам;
   - тарифы -15%;
   - перерасход CAPEX +20%.
   Для каждого фактора укажи пересчитанные NPV и IRR и оцени критичность.

4. Проверка внутренней согласованности модели — используй таблицу из шага 1. Если найдены расхождения — зафиксируй их.

5. Оценка рисков для бюджетных средств — доля субсидии в CAPEX, зависимость проекта от бюджетной поддержки, что произойдёт при пессимистичном сценарии.

6. Заключение и рекомендация — одна из трёх формулировок: «поддержать», «поддержать с условиями», «отклонить». Обоснуй выбор, ссылаясь на критерии из раздела 3 регламента. Учти, что NPV в пессимистичном сценарии отрицательный — укажи условия, при которых проект целесообразен (согласно разделу 4 регламента).


Шаг 3. Визуализация ключевых показателей

На основе рассчитанных данных построй три Mermaid-диаграммы для включения в заключение:

1. Столбчатая диаграмма NPV по трём сценариям — возьми значения NPV из таблицы сценариев, которую ты составил в шаге 2, и визуализируй их (формат xychart-beta).

2. Столбчатая диаграмма чувствительности IRR — возьми результаты анализа чувствительности из шага 2 и покажи, как меняется IRR при каждом из 4 стресс-факторов.

3. Визуализация движения денежных средств по годам базового сценария — покажи, в какие годы свободный CF отрицательный, а в какие положительный.

Все цифры для диаграмм бери из своих расчётов, не из исходной модели напрямую.
```
