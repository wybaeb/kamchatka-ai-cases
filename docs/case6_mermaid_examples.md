## Пример 2: Процесс рассмотрения обращения гражданина

```mermaid
flowchart TD
    A[Поступление обращения] --> B{Тип обращения}
    B -->|Жалоба| C[Регистрация в СЭД]
    B -->|Предложение| C
    B -->|Заявление| C
    C --> D[Определение исполнителя]
    D --> E{Срок по 59-ФЗ}
    E -->|Стандартное| F[30 дней]
    E -->|Срочное| G[15 дней]
    E -->|Особый контроль| H[7 дней]
    F --> I[Подготовка ответа]
    G --> I
    H --> I
    I --> J[Согласование руководителем]
    J --> K[Направление ответа заявителю]
    K --> L[Снятие с контроля]

    style A fill:#1e3a5f,stroke:#4a90d9
    style B fill:#2d1b4e,stroke:#8b5cf6
    style K fill:#1a4d2e,stroke:#22c55e
    style L fill:#1a4d2e,stroke:#22c55e
```

## Пример 3: Gantt-диаграмма — план мероприятий северного завоза

```mermaid
gantt
    title План северного завоза Камчатского края, 2026
    dateFormat  YYYY-MM-DD
    axisFormat  %b

    section Морской транспорт
    Рейс 1 (Владивосток — Петропавловск)   :done, m1, 2026-01-15, 2026-02-05
    Рейс 2 (продовольствие)                :done, m2, 2026-03-01, 2026-03-20
    Рейс 3 (стройматериалы)                :active, m3, 2026-04-10, 2026-05-01
    Рейс 4 (топливо, летний)               :m4, 2026-06-01, 2026-06-25
    Рейс 5 (основной осенний)              :m5, 2026-08-15, 2026-09-10

    section Авиадоставка
    Экстренный завоз (Тиличики)            :crit, a1, 2026-04-05, 2026-04-12
    Плановый завоз (Палана)                :a2, 2026-04-15, 2026-04-22

    section Автозимники
    Эксплуатация зимника Эссо-Палана       :done, z1, 2026-01-01, 2026-03-31
    Эксплуатация зимника Ключи-Тиличики    :done, z2, 2026-01-15, 2026-03-15
```

## Пример 4: Организационная структура (фрагмент)

```mermaid
graph TD
    GOV[Губернатор Камчатского края] --> VP1[Зам. председателя<br>Экономика и финансы]
    GOV --> VP2[Зам. председателя<br>Социальная политика]
    GOV --> VP3[Зам. председателя<br>Инфраструктура]

    VP1 --> MIN1[Минэкономразвития]
    VP1 --> MIN2[Минфин]
    VP1 --> AG1[Агентство по туризму]

    VP2 --> MIN3[Минобразования]
    VP2 --> MIN4[Минздрав]

    VP3 --> MIN6[Минстрой]
    VP3 --> MIN7[Минтранспорт]
```

## Пример 5: Sequence-диаграмма — межведомственное согласование

```mermaid
sequenceDiagram
    participant МЭР as Минэкономразвития
    participant МФ as Минфин
    participant АТ as Агентство по туризму
    participant ГУБ as Губернатор

    МЭР->>АТ: Запрос показателей турпотока
    АТ-->>МЭР: Данные за квартал
    МЭР->>МФ: Проект бюджетной заявки
    МФ-->>МЭР: Замечания по обоснованию
    МЭР->>МФ: Доработанная заявка
    МФ->>ГУБ: Сводный доклад
    ГУБ-->>МФ: Утверждение с поручениями
    МФ->>МЭР: Доведение лимитов
    МЭР->>АТ: Уведомление о финансировании
```
