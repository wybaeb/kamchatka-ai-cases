# Промпт для ИИ-ассистента — Кейс 2

## Задача: Анализ и сравнение сценариев финансовой модели

---

### Полный промпт для DeepSeek / GigaChat

Ты — финансовый аналитик, готовящий заключение для инвестиционного комитета Правительства Камчатского края.

Тебе передана финансовая модель проекта «Всесезонный туристический комплекс "Вилючинский"» с тремя сценариями.

**Шаг 1. Сравнительная таблица сценариев.**
Верни таблицу:
- показатель (NPV, IRR, срок окупаемости, пиковый кассовый разрыв, кумулятивный свободный CF за 10 лет),
- сценарий A (базовый),
- сценарий B (оптимистичный),
- сценарий C (пессимистичный),
- комментарий (1 предложение).

**Шаг 2. Анализ рисков.**
Для каждого сценария определи:
- год, в котором наступает кассовый разрыв (если есть),
- максимальную потребность в дополнительном финансировании,
- зависимость от субсидии (что будет, если субсидия не поступит вовремя),
- чувствительность IRR к ставке кредита (при +2 п.п.).

**Шаг 3. Проверка логики модели.**
Проверь внутреннюю согласованность:
- совпадают ли CAPEX и источники финансирования,
- согласована ли выручка с загрузкой и тарифами,
- корректно ли рассчитан OPEX как % от выручки,
- нет ли арифметических ошибок.

**Шаг 4. Заключение.**
Сформулируй краткое заключение (5–7 предложений):
- какой сценарий наиболее вероятен и почему,
- при каких условиях проект становится убыточным,
- рекомендация: поддержать / доработать / отклонить.
