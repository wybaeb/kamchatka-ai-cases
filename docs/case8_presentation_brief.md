# Поручение заместителя Председателя Правительства Камчатского края

**Документ для практики — Кейс 8 (подготовка презентации на основе квартального доклада)**

---

**Исх. № 214-ЗП от 18.04.2026**

Заместитель Председателя Правительства Камчатского края — руководителю Департамента аналитического обеспечения

На основании подготовленного квартального доклада о социально-экономическом развитии Камчатского края за I квартал 2026 года подготовьте презентацию из 7–10 слайдов для расширенного заседания Правительства, назначенного на 22 апреля 2026 года.

**Требования к презентации:**

1. Первый слайд — заголовок и ключевые цифры квартала (ВРП, турпоток, бюджет).
2. По каждому из основных разделов (экономика, бюджет, социальная сфера, туризм, инфраструктура) — один слайд с 3–4 тезисами и визуализацией (график или таблица).
3. Отдельный слайд «Проблемные зоны» — таблица с показателями, имеющими отклонения от плана, с указанием причин.
4. Слайд «Северный завоз» — детализация ситуации с задержками поставок, число затронутых населённых пунктов, предлагаемые меры по нормализации.
5. Финальный слайд — три ключевых вывода и предложения Правительству.

**Формат подготовки:** подготовьте сначала структуру презентации в формате Markdown, затем загрузите готовый файл в сервис Gamma App (https://gamma.app) для генерации визуальной презентации. Для редактирования Markdown используйте MarkText (https://github.com/marktext/marktext) или любой текстовый редактор.

Срок представления: до 12:00 21 апреля 2026 года.
