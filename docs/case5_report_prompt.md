# Промпт для ИИ-ассистента — Кейс 5

## Задача: Формирование квартального отчёта с аналитическими выводами

---

### Полный промпт для DeepSeek / GigaChat

Ты — аналитик Министерства экономического развития Камчатского края. Тебе передана табличная выгрузка с показателями за I квартал 2026 года.

**Шаг 1. Текст отчёта.**
Подготовь структурированный текст квартального отчёта для доклада заместителю Председателя Правительства Камчатского края. Формат:
- заголовок,
- краткое резюме (3-4 предложения — ключевые тенденции квартала),
- по каждому разделу (экономика, бюджет, соцсфера, туризм, инфраструктура, северный завоз):
  - 2-3 факта с цифрами,
  - аналитический вывод: динамика, причины, отклонения от плана,
  - рекомендация или сигнал внимания (если есть проблема).
- заключение с 3 ключевыми выводами.

**Шаг 2. Mermaid-визуализация.**
Создай Mermaid-диаграмму (тип pie или bar — на твоё усмотрение), отражающую структуру доходов бюджета (собственные vs трансферты) и динамику ключевых показателей.

**Шаг 3. Проблемные зоны.**
Отдельным блоком выдели показатели, по которым наблюдается ухудшение или отставание от плана. Для каждого укажи:
- показатель,
- факт vs план / прошлый год,
- степень отклонения,
- возможная причина,
- предлагаемое действие.

Стиль: деловой, без оценочных суждений, факты отделены от гипотез.
