# Промпт — Кейс 3

```text
Тебе переданы три документа:
- Действующая редакция порядка предоставления субсидий СМСП в сфере туризма (`case3_regulation_v1.md`);
- Проект изменений в данный порядок (`case3_regulation_v2.md`);
- Техническое задание на подготовку аналитической справки (`case3_analysis_brief.md`).

Выполни задание в два шага.


Шаг 1. Сравнительная таблица изменений

Составь полную таблицу всех изменений между действующей редакцией и проектом. Для каждого изменения укажи:

| № | Пункт | Было (действующая редакция) | Стало (проект изменений) | Тип изменения |
|---|-------|-----------------------------|--------------------------|---------------|

Правила заполнения:
- В столбце «Тип изменения» укажи: редакционное (уточнение формулировки без изменения смысла) или смысловое (изменение правового регулирования).
- Включи все пункты, в которых есть отличия, в том числе полностью новые пункты (для них в столбце «Было» укажи «отсутствует»).
- Если в одном пункте несколько независимых изменений — выдели каждое отдельной строкой.


Шаг 2. Аналитическая справка

На основании таблицы из Шага 1 и требований из технического задания подготовь аналитическую справку для заместителя Председателя Правительства Камчатского края. Справка должна включать:

1. Ключевые изменения — описание наиболее существенных смысловых изменений, сгруппированных по четырём категориям:
   - По кругу получателей
   - По объёму финансирования
   - По процедуре
   - По контролю и отчётности

2. Оценка рисков — какие изменения могут создать проблемы при правоприменении (неоднозначное толкование, правовые коллизии, риски злоупотреблений, организационные и технические сложности).

3. Рекомендации — конкретные замечания и предложения, которые целесообразно направить разработчику проекта.

Требования к справке:
- Объём — не более 3 страниц (без учёта таблицы).
- Стиль — официально-деловой.
- Каждый тезис должен быть подкреплён ссылкой на конкретный пункт нормативного акта.

Шаг 3. Визуализация изменений (HTML-diff)

Подготовь HTML-документ, в котором полный текст нормативного акта отображается с визуальной разметкой изменений — аналогично режиму отслеживания правок (track changes):
- Удалённый текст — красный фон, зачёркивание: <span style="background:#ffe5e5;color:#a12638;text-decoration:line-through">удалённый текст</span>
- Добавленный текст — зелёный фон: <span style="background:#e7f7e7;color:#2f7d32">добавленный текст</span>
- Неизменённый текст — без выделения.

Результат — самодостаточный HTML-файл, который можно открыть в браузере для визуальной сверки.
```
