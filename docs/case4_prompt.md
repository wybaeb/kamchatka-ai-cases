# Промпт — Кейс 4

```text
Тебе переданы два документа:
1. case4_risk_indicators.md — перечень из 10 индикаторов риска нарушения обязательных требований в области экологического контроля Камчатского края, включая контекст региональных особенностей.
2. case4_risk_methodology.md — методические рекомендации по верификации индикаторов с критериями оценки и форматом таблицы.

Шаг 1. Верификация действующих индикаторов.
Для каждого из 10 индикаторов заполни таблицу верификации по формату из методрекомендаций: оцени по 4 критериям (0-2 балла), определи итоговый статус и рекомендацию.

Шаг 2. Предложения по модификации.
Для индикаторов со статусом «нерабочий» или «требует модификации» предложи конкретные изменения: новую формулировку, порог, источник данных.

Шаг 3. Новые индикаторы.
Предложи 3-5 новых индикаторов, учитывающих камчатскую специфику: ООПТ, нерестовые реки, геотермальные зоны, КМНС, сезонность.
```
