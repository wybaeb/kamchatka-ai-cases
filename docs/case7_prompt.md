# Промпт — Кейс 7

```text
Тебе переданы три документа:

1. Табличная выгрузка показателей (`case7_quarterly_data.md`) — статистические данные о социально-экономическом развитии Камчатского края за I квартал 2026 года.
2. Шаблон отчёта (`case7_report_template.md`) — утверждённая структура ежеквартального доклада с требованиями к содержанию каждого раздела.
3. Поручение заместителя Председателя (`case7_head_requirements.md`) — указания руководства с акцентами, которые необходимо отразить в докладе.

DeepSeek и GigaChat умеют строить Mermaid-диаграммы из данных — используй это для визуализации в докладе.


Шаг 1. Анализ данных

Проанализируй табличную выгрузку (`case7_quarterly_data.md`). Выяви:
- показатели с положительной динамикой;
- показатели с отрицательной динамикой или существенным отклонением от плана;
- взаимосвязи между показателями (например, рост инвестиций и рост строительства; снижение северного завоза и рост задержек).

Сформулируй 3–5 ключевых наблюдений, которые должны лечь в основу доклада.


Шаг 2. Подготовка доклада по шаблону

На основе проведённого анализа подготовь полный текст доклада, строго следуя структуре из шаблона (`case7_report_template.md`):

- Раздел 1 (Резюме): 3–4 предложения с ключевыми тенденциями.
- Раздел 2 (Экономика): все подразделы — ВРП, промпроизводство, рыбодобыча, строительство, торговля, инвестиции. Для каждого: факт, план, отклонение, аналитический вывод.
- Раздел 3 (Бюджет): доходы (с разбивкой на собственные и трансферты), расходы, дефицит, госдолг. Включить круговую диаграмму (pie) структуры доходов.
- Раздел 4 (Социальная сфера): зарплата, безработица, демография.
- Раздел 5 (Туризм): турпоток, загрузка, объём услуг.
- Раздел 6 (Инфраструктура и северный завоз): транспорт, жильё, газификация, северный завоз.
- Раздел 7 (Проблемные зоны): таблица с показателями, имеющими отрицательное отклонение — факт, план, отклонение, причина, предлагаемое решение.
- Раздел 8 (Выводы и предложения): 3–5 конкретных пунктов.

Формулировки — нейтральные, без оценочных суждений. Гипотезы явно отделены от фактов. Все цифры — из выгрузки.


Шаг 3. Учёт акцентов из поручения

Перечитай поручение заместителя Председателя (`case7_head_requirements.md`) и убедись, что в докладе:

1. Северный завоз — подробно раскрыты причины задержек и перебоев, указан масштаб (число пострадавших населённых пунктов, рост средней задержки), предложены конкретные экстренные меры.
2. Турпоток — дана оценка, обеспечивается ли целевой рост 18%. Если фактический рост ниже целевого — указаны причины отставания и корректирующие мероприятия.
3. Дефицит бюджета — объяснены причины увеличения дефицита, дана оценка рисков для исполнения расходных обязательств.
4. Визуализация — построй из данных выгрузки не менее двух Mermaid-диаграмм:
   (а) pie-диаграмма структуры доходов бюджета (налоговые / неналоговые / трансферты — рассчитай доли из цифр);
   (б) xychart-beta столбчатая диаграмма динамики турпотока I кв. 2025 vs I кв. 2026 vs план 2026.
   Диаграммы строятся из цифр выгрузки, не из шаблонов.

Если какой-либо акцент из поручения раскрыт недостаточно, дополни соответствующий раздел.

Шаг 4. VBA-макрос для автоматизации

Подготовь VBA-макрос для Excel, который:
a) создаёт лист «Сводка» и копирует на него ключевые показатели из исходной таблицы;
b) рассчитывает отклонение факт/план в процентах для каждого показателя;
c) подсвечивает красным (RGB 255, 200, 200) ячейки с отклонением хуже -5%;
d) подсвечивает зелёным (RGB 200, 255, 200) ячейки с отклонением лучше +5%;
e) добавляет итоговую строку «Количество показателей с отклонением».

Макрос должен быть готов к вставке в редактор VBA (Alt+F11 → Insert Module → вставить код).
```
