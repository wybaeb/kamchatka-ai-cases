# Промпт для ИИ-ассистента — Кейс 4

## Задача: Верификация и разработка индикаторов риска

---

### Полный промпт для DeepSeek / GigaChat

Ты — методолог контрольно-надзорной деятельности. Тебе передан перечень индикаторов риска нарушения обязательных требований в области экологического контроля на территории Камчатского края, а также методические рекомендации и контекст региональных особенностей.

**Шаг 1. Верификация каждого индикатора.**
Для каждого из 10 индикаторов заполни строку таблицы:
- номер индикатора,
- соответствие критерию (а) — доступность данных: да / частично / нет,
- соответствие критерию (б) — чёткость порога: да / размытый,
- соответствие критерию (в) — прогностическая ценность: высокая / средняя / низкая,
- соответствие критерию (г) — верифицируемость: да / частично / нет,
- итоговый статус: рабочий-эффективный / рабочий-неэффективный / нерабочий,
- рекомендация: оставить / модифицировать / исключить,
- обоснование (1-2 предложения с учётом камчатской специфики).

**Шаг 2. Предложения по модификации.**
Для каждого индикатора со статусом «модифицировать» предложи конкретные изменения:
- новая формулировка,
- новый порог срабатывания,
- альтернативный источник данных (если текущий недоступен).

**Шаг 3. Новые индикаторы.**
Предложи 3–5 новых индикаторов, учитывающих специфику Камчатского края:
- формулировка,
- источник данных,
- порог срабатывания,
- обоснование прогностической ценности.

Учитывай: ООПТ, нерестовые реки, геотермальные зоны, КМНС, сезонность, труднодоступность.
