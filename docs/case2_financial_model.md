# Финансовая модель инвестиционного проекта

**Документ для практики — Кейс 2 (анализ финансовых моделей)**

---

## Проект: Всесезонный туристический комплекс «Вилючинский»

**Инициатор:** АО «Корпорация развития Камчатского края»
**Локация:** подножие вулкана Вилючинский, Елизовский район
**Площадь участка:** 12,4 га
**Горизонт планирования:** 10 лет (2027–2036)

---

### 1. Капитальные затраты (CAPEX)

| Статья | Сумма, млн руб. |
|---|---|
| Проектирование и изыскания | 45 |
| Подготовка территории (дороги, коммуникации) | 320 |
| Строительство гостиничного комплекса (120 номеров) | 680 |
| Строительство термального спа-комплекса | 180 |
| Горнолыжная инфраструктура (2 подъёмника, трассы) | 450 |
| Визит-центр и ресторан | 95 |
| Благоустройство и экотропы | 60 |
| Оборудование и мебель | 140 |
| Непредвиденные расходы (10%) | 197 |
| **Итого CAPEX** | **2 167** |

**Источники финансирования:**
- Собственные средства инвестора: 650 млн руб. (30%)

- Субсидия из федерального бюджета (программа «Развитие туризма»): 517 млн руб. (24%)

### 4. Ключевые риски (идентифицированные)

| Риск | Вероятность | Влияние | Комментарий |
|---|---|---|---|
| Задержка строительства из-за логистики | Высокая | Среднее | Зависимость от северного завоза стройматериалов |
| Недозагрузка в низкий сезон | Средняя | Высокое | Конкуренция с южными направлениями |
| Рост ключевой ставки | Средняя | Высокое | При 16%+ проект убыточен |
| Сейсмическая активность | Низкая | Критическое | Страхование увеличит OPEX на 3–5% |
| Ограничения ООПТ | Средняя | Среднее | Часть территории может попасть в буферную зону |
