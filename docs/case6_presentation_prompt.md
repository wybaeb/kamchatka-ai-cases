# Промпт для ИИ-ассистента — Кейс 6

## Задача: Подготовка доклада и структуры презентации

---

### Полный промпт для DeepSeek / GigaChat

Ты — помощник руководителя Агентства по туризму Камчатского края. Тебе переданы черновые заметки, таблица показателей и выдержки из поручений.

**Шаг 1. Структура доклада.**
Подготовь текст доклада на 10 минут (~ 1 500 слов) для расширенного заседания Правительства. Структура:
1. Вступление — 1 мин (контекст, место отрасли в экономике края)
2. Итоги 2025 — 3 мин (ключевые достижения, цифры)
3. Проблемы и вызовы — 2 мин (дефициты, экология, сезонность)
4. Задачи на 2026 — 3 мин (приоритеты, ресурсы, сроки)
5. Заключение — 1 мин (ключевой месседж для Губернатора)

Стиль: чёткий, управленческий, факты + выводы, без воды.

**Шаг 2. Разбивка на слайды.**
Предложи структуру презентации из 5–7 слайдов:
- заголовок слайда,
- 3-4 тезиса (что на слайде),
- тип визуализации (график, таблица, диаграмма, текст).

**Шаг 3. Адаптация под аудиторию.**
Подготовь три варианта ключевого слайда «Итоги 2025»:
- для Губернатора (стратегический фокус, KPI, поручения),
- для экспертного сообщества (методология, данные, сравнения),
- для публичного выступления (простые метафоры, визуальные образы).

Для каждого варианта: заголовок, 4 тезиса, рекомендация по визуализации.
